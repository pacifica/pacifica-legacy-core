Every file uploaded to MyEMSL gets a unique itemid. You can retrieve the itemid from the query system. To find out more information about a particular item, you can use the iteminfo service. To do so, append the itemid and the string "/xml" to the "iteminfo" service url and perform a get request.

The resulting format will resemble:

<?xml version="1.0"?>
<myemsl>
   <itemid>46</itemid>
   <type>file</type>
   <filename>Clean Phenakite_001.tif</filename>
   <size>976172</size>
   <verified>False</verified>
   <aged>False</aged>
   <submitter first="Joe" last="Bob" id="39822"/>
   <stime>2011-07-28 15:54:26.655496</stime>
   <checksum>
      <sha1>70f6f6bbb67203eaaf305625a177270f229ac43a</sha1>
   </checksum>
</myemsl>
