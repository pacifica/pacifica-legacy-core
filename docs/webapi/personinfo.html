To find out more information about the currently logged in user, make a get request to the "personinfo" service with "/xml" appended. The result will resemble the following:

<?xml version="1.0"?>
<myemsl>
   <personid>39821</personid>
   <firstname>Joe</firstname>
   <lastname>Smith</lastname>
   <emailaddress>Joe.Smith@pnnl.gov</emailaddress>
   <networkid>joesmith</networkid>
</myemsl>
