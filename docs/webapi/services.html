The service description file format is an xml document. It looks like:

<?xml version="1.0"?>
<myemsl>
	<prefix>http://url</prefix>
	<services>
		<service name="service1" location="/service1/" />
		<service name="service2" location="/service2/" />
		...
		<service name="logout" location="/logout/" />
	</services>
</myemsl>

If a service's location starts with "/", the value of prefix must be prepended to it when processing.

Every MyEMSL instance must run at least one copy of this service. By default it can be retrieved from "<INSTANCE_URL>/myemsl/services/"
