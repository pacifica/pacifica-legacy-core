First, all services, unless otherwise specified, require an authenticated session. To get one, follow the directions in "protocols/authentication.html". Once you have an authenticated session, you can use the webapi to access MyEMSL services. As a reminder, please call the "logout" service when done with your session.

Some helpful hints for using web services are here:

  1. To test if your session is valid after authentication, see "webapi/testauth.html"
  2. If you wish to upload data into MyEMSL, without using existing uploaders, have a look at "format/bundle.html" and "webapi/ingest.html"
  3. If you wish to search through the metadata, see "webapi/elasticsearch.html"
  4. To retrieve extended information about a file see "webapi/iteminfo.html"
  5. To retrieve extended information about your user, see "webapi/personinfo.html"
  6. To retrieve data:
    1. Use the cart api "webapi/cart.html"
    2. Or for single files "webapi/item.html"

Some helpful hints for integrating sites or services into MyEMSL are here:

  1. If you wish to plug in a basic site or service into an existing MyEMSL server, see "plugins/basicwebsite.html"
