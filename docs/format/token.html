A token is a base64 - '\n' - '=':

  1. An ascii base 10 formatted length of public section.
  2. A json encoded document that has been modified:
    1. The leading '{' has been removed. It will automatically be re-added.
    2. The tail '}' has been removed. It will automatically be re-added.
  3. An optional additional data section. If specified, its size must be specified in an 'o':<len> entry in the root of #2.
  4. The signature from the authorization authority.
