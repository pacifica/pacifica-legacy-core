Minimum amount of timeThis value will specify the minimum amount of time that the system will use to decide when to refresh values from the URL in hours. If the value is 0, the system could decide to always download the list on every request. Setting it greater then one ensures that the value is cached for at minimum the specified number of hours. Requested timeThis value specifies how often you recommend the system downloads the valid values from the server. Setting it to 0 means that the system will never refresh once it is downloaded for the first time. For non zero values, the value is only a recommendation and the system can choose to do it at a more opertune rate.
